Installation Guide

Get SplitX running on your Android device in 5 easy steps

1

Download the APK

Click the "Download APK" button above to download the latest splitx.apk file to your device. The file is about 12 MB.

💡 If Chrome shows a warning that the file "can be harmful", tap "Download anyway" — this is normal for APK files outside the Play Store.
2

Enable Unknown Sources

Android blocks installs from sources outside the Play Store by default. You need to allow it once:

⚙️ Go to Settings → Security → Install unknown apps → select your browser (Chrome) → toggle "Allow from this source" ON.
📌 Alternate path (Android 12+): Settings → Apps → Special app access → Install unknown apps → Chrome → Allow
3

Install the APK

Open your Downloads folder (or tap the download notification) and tap on splitx.apk. Press "Install" when prompted.

⏱️ Installation takes about 5-10 seconds. You'll see "App installed" when done.
4

Open & Create Account

Launch SplitX from your app drawer. You'll see a beautiful welcome walkthrough — swipe through it and then sign up with your name, email, and password.

🎉 Your account is created instantly. Notifications are auto-subscribed — you'll get real-time updates right away!
5

Start Splitting!

Create your first group, invite friends using the group code, and start adding expenses. SplitX handles the math — you handle the fun! 🎯

Frequently Asked Questions

Got questions? We've got answers.

Is SplitX free to use?
Yes! SplitX is completely free. No ads, no hidden fees, no premium tier. All features are available to everyone.
Is it safe to install APK files?
Absolutely. Our APK is the same code that would be on the Play Store. We use HTTPS for downloads and your data is protected with JWT encryption and bcrypt password hashing.
How do I update the app?
Simply come back to this page and download the latest APK. Install it over the existing app — your data will be preserved. We'll also send an in-app notification when updates are available.
Does it work on iOS / iPhone?
Currently SplitX is available for Android only. iOS support is coming soon! In the meantime you can use the web version at this URL.
My phone says "Blocked by Play Protect"
Google Play Protect sometimes warns about apps installed outside the Play Store. Tap "Install anyway" or "More details → Install without scanning". This is standard for all sideloaded APKs and is safe.
Can I use it without creating an account?
An account is required so your expenses, groups, and settlements can be synced and shared with other users. Signup takes just 10 seconds!
